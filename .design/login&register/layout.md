# 登录与注册原型图解读

基于提供的 Figma 导出数据（CSS 和布局数据），以下是对“登录”和“注册”页面原型的解读。设计采用了居中的垂直堆叠布局，使用不同的矩形块代表交互元素。

## 1. 登录页面 (`proto/login/login.md`)

登录页面设计为一个简单的居中卡片或垂直流，包含以下元素：

### 布局结构
- **容器**: 1728x1117px 的白色背景画布 (`#FFFFFF`)。
- **对齐**: 所有元素相对于主要内容区域水平居中（大约在 x=634px 处）。

### 元素解读

| 元素 ID | 视觉外观 | 上下文/标签 | 解读 / 功能 |
| :--- | :--- | :--- | :--- |
| **Rectangle 21** | 大灰色方块 (459x181px) | 包含文本 "Sign in to EduHub" | **页面标题 / Logo 区域**。这是表单的主标题容器。 |
| **Rectangle 22** | 中灰色方块 (459x105px) | 位于 "Username" 标签下方 | **用户名输入框**。用户在此输入用户名或邮箱。 |
| **Rectangle 23** | 中灰色方块 (459x105px) | 位于 "Password" 标签下方 | **密码输入框**。用户在此输入密码。 |
| **Rectangle 24** | 中灰色方块 (459x105px) | 包含文本 "Sign In" | **登录按钮**。提交凭据的主要操作按钮。 |
| **Text Link** | 纯文本 | "New to EduHub? Create an account" | **导航链接**。将用户重定向到注册页面。 |

## 2. 注册页面 (`proto/register/register.md`)

注册页面遵循与登录页面完全相同的设计语言，但增加了一个密码确认字段。

### 布局结构
- **容器**: 尺寸和背景与登录页面相同。
- **对齐**: 一致的垂直堆叠。

### 元素解读

| 元素 ID | 视觉外观 | 上下文/标签 | 解读 / 功能 |
| :--- | :--- | :--- | :--- |
| **Rectangle 21** | 大灰色方块 (459x181px) | 包含文本 "Register to EduHub" | **页面标题 / Logo 区域**。注册表单的主标题容器。 |
| **Rectangle 22** | 中灰色方块 (459x105px) | 位于 "Username" 标签下方 | **用户名输入框**。用户选择他们的用户名。 |
| **Rectangle 23** | 中灰色方块 (459x105px) | 位于 "Password" 标签下方 | **密码输入框**。用户设置他们的密码。 |
| **Rectangle 25** | 中灰色方块 (459x105px) | 位于 "Password Confirm" 标签下方 | **确认密码输入框**。用户重新输入密码以确保准确性。 |
| **Rectangle 24** | 中灰色方块 (459x105px) | 包含文本 "Register" | **注册按钮**。创建账户的主要操作按钮。 |
| **Text Link** | 纯文本 | "Have account already? Login in to EduHub" | **导航链接**。将用户重定向回登录页面。 |

## 共享设计系统总结
- **输入框**: 由 459x105px 的灰色矩形 (`#D9D9D9`) 表示。
- **标题**: 由较高的 459x181px 灰色矩形表示。
- **排版**: 使用 'Inter' 字体，24px 大小，黑色 (`#000000`)。
- **间距**: 标签和输入框之间保持一致的垂直间距。
